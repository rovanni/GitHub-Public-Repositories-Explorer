Sobre

Olá, Eu Sou Luciano Rovanni do Nascimento

Seja bem-vindo à minha página! Sou professor de programação web e estou aqui para ajudar você a aprender como criar páginas web responsivas e acessíveis para acessar a API do GitHub.

O Que Você Vai Aprender

Nesta jornada de aprendizado, você irá:

  • Aprender a criar páginas web responsivas que se adaptam a diferentes dispositivos, como celulares e tablets.
  • Ganhar experiência em acessibilidade web, tornando suas páginas acessíveis a todas as pessoas, independentemente de suas habilidades ou necessidades.
  • Explorar como acessar a API do GitHub para consultar os Repositórios Públicos de usuários do GitHub.

Por Que Isso é Importante

A tecnologia desempenha um papel fundamental em nossas vidas e carreiras. Aprender programação web e desenvolver habilidades de criação de páginas responsivas e acessíveis é essencial para se destacar no mundo digital de hoje.

Com as habilidades que você adquirir aqui, você estará preparado para criar sites e aplicativos web incríveis e contribuir para a construção de um mundo digital mais inclusivo.

Seu Sucesso é Meu Objetivo

Meu objetivo como professor é ajudar você a alcançar o sucesso. Estou aqui para apoiar sua jornada de aprendizado e responder a todas as suas perguntas. Juntos, podemos explorar o emocionante mundo da programação web e da tecnologia.
